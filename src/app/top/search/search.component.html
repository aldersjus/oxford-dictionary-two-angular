Oxford English Dictionary application.

This has been built using the Angular framework. It is a simple application that connects to the Oxford English dictionary API to search for the entered word. It is not production ready, it was built to learn the basics of working with Angular and using the HTTP library.


How to use.

Remember that you will need your own API key and password from Oxford English Dictionary. Enter a word in the box and the results will be displayed here. It currently only works with the base form of verbs, so if enter you "walking" instead of "walk" it might fail to return anything, this application will display any errors in an alert box.
